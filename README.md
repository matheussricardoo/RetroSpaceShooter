### 📁 Project Structure | Estrutura do Projeto

```
/
├── index.html
├── styles.css
├── sprites.js      # Pixel art sprites and rendering
├── stars.js        # Star field background
├── player.js       # Player ship controls
├── enemy.js        # Enemy behavior
├── bullet.js       # Projectile system
├── game.js         # Main game logic
└── translations.js # Language support
```

### 🚀 Getting Started | Começando

```bash
# Clone the repository | Clone o repositório
git clone https://github.com/matheussricardoo/retrospaceshooter.git

# Navigate to project directory | Navegue até o diretório do projeto
cd retrospaceshooter

# Open in browser | Abra no navegador
# You can use any local server, for example:
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```
